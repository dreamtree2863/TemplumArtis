내 Google Drive 음악을 싱크 가사와 함께.

최초 1회만. 설정 방법은 아래 도움말 참고.

OAuth 클라이언트 ID 만드는 법
  1. console.cloud.google.com → 프로젝트 생성
  2. "API 및 서비스" → 라이브러리 → Google Drive API 사용 설정
  3. "OAuth 동의 화면" 구성(외부, 테스트 사용자에 본인 계정 추가)
  4. "사용자 인증 정보" → OAuth 클라이언트 ID → 웹 애플리케이션
  5. 승인된 자바스크립트 원본에 이 페이지 주소(예: https://아이디.github.io) 추가
  6. 생성된 클라이언트 ID를 위에 붙여넣기
